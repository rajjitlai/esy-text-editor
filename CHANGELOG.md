# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-06-05

### Added

- React + Electron application scaffold for Esy Text Editor.
- Markdown-first editor with live preview.
- Workspace sidebar for folder browsing and file loading.
- Tabbed document workflow with basic open/save support.
- Monochrome shadcn-style UI built with Tailwind CSS.
- Electron Builder configuration for Windows, macOS, and Linux packages.

### Changed

- Replaced the old C/X11 direction with a desktop web stack.
- Reworked the README to reflect the new architecture and workflow.
