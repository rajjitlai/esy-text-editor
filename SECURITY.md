# Security Policy

## Supported Versions

Security fixes are tracked for the latest released version of Esy Text Editor.

## Reporting a Vulnerability

Do not open a public issue for a security problem.

Report suspected vulnerabilities privately to the project maintainer or repository owner with:

- A short summary of the issue
- Reproduction steps
- Affected version or commit
- Any proof of concept or logs you have

## Scope

Security issues include IPC exposure, filesystem access, packaging misconfiguration, and dependency vulnerabilities.
