# Code of Conduct

## Our Pledge

We expect contributors to act with respect, clarity, and professionalism.

## Standards

- Be direct without being abusive.
- Challenge ideas, not people.
- Keep feedback technical and specific.
- Do not tolerate harassment, discrimination, or personal attacks.

## Enforcement

Project maintainers may remove content or contributions that violate this code of conduct.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, and discussion channels.
